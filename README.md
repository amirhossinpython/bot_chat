# bot_chat
این پروژه یک ربات روبیکا هست که امکان چت با هوش مصنوعی رو فراهم می‌کنه. در بخش متغیر system_prompt شما می‌تونید پرامپت دلخواه خودتون رو وارد کنید تا ربات بر اساس اون رفتار کنه و پاسخ بده.  یعنی کافیه پرامپت مخصوص به موضوع یا سبک مدنظرتون رو بذارید تا ربات دقیقا همون‌طور با کاربرها چت کنه.
